GYMFLOW – HTML TĨNH DÀNH CHO FIGMA

1. Mở index.html để xem danh sách toàn bộ màn hình.
2. Mỗi file HTML là một snapshot độc lập, không chứa JavaScript.
3. CSS GymFlow đã được nhúng trực tiếp vào từng file.
4. Có thể kéo file vào html.to.design/html.io hoặc mở file rồi capture.
5. Font Inter và Bootstrap Icons được tải qua CDN, vì vậy nên giữ kết nối Internet khi import.
6. Chạy scripts\export-figma-html.ps1 từ thư mục dự án để tái xuất sau khi giao diện thay đổi.
